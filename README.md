# OSI vs TCPIP (SOC Perspective)
Understanding OSI vs TCP/IP models from a SOC Analyst perspective with attack mapping and detection insights

## Why This Matters for SOC
Understanding which attacks occur at which network layer helps analysts:
- Identify attack patterns faster
- Correlate alerts correctly
- Analyze logs and PCAPs efficiently

## What This Repo Covers
- OSI 7 Layers 
- TCP/IP 4 Layer Model
- Attack mapping per layer
- SOC detection perspective

## Target Role
- SOC Analyst L1 / L2
- Blue Team
- Incident Response

## Tools Referenced
- Wireshark
- Firewall Logs
- IDS/IPS
- Proxy Logs
